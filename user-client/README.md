## user-client

user-client - клиент для блога.
### Доступные страницы

 - **/**          - Главная страница, сслыка на соц сети и переход к постам.
 - **/blog**      - Список постов, с пагинацией.
 - **/blog/{id}** - Страница отдельного поста.
